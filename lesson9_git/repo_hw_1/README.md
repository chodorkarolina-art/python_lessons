# repo_hw_1
Zadanie domowe 1: Zespołowe tworzenie małego projektu

## Struktura projektu

- main – gałąź produkcyjna
- develop – gałąź rozwojowa
- src – kod źródłowy
- tests – testy
- docs – dokumentacja

## Konfiguracja ochrony gałęzi main

Skonfigurowano ochronę gałęzi main:
- wymagane 2 zatwierdzenia (reviewers)
- wymagany Pull Request przed mergem
- wymagane przejście status checks
- zablokowane bezpośrednie commity do main
- zablokowane usuwanie i force push
